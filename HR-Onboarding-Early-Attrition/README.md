# HR Onboarding & Early Attrition Analysis

## Project Overview
This project analyzes employee onboarding data to identify early attrition trends and improve employee retention.

## Objective
Identify patterns in employee exits within the first 6 months and provide insights to improve onboarding experience.

## Tools Used
- Excel
- Power BI
- Data Analysis

## Key Insights
- Attrition trends during the first 90 days
- Department-wise attrition patterns
- Potential improvements in onboarding processes

## Outcome
Provided recommendations to improve onboarding effectiveness and reduce early attrition.
