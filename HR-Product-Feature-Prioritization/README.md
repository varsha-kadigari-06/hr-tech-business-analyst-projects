# HR Product Feature Prioritization

## Project Overview
This project focuses on prioritizing HR product features based on business impact and user needs.

## Objective
Identify and prioritize features for an HR technology platform to improve employee experience and HR efficiency.

## Tools Used
- Excel
- Business Analysis Framework

## Approach
Used prioritization techniques such as:
- Impact vs Effort matrix
- Stakeholder requirement analysis

## Outcome
Created a structured prioritization framework to guide HR product feature development.
